# Images

Place the following images in this directory:

- `hero-bg.jpg` — Landing page hero background (warm candlelight/floral, 1920×1080 minimum)
  - Use: soft desaturated warmth, amber/gold tones
  - Source: Unsplash (search "candles warm", "golden hour flowers")
  
- `default-memorial-art.jpg` — Fallback when DALL-E is not configured (600×600)
  - A warm abstract watercolor

- `og-default.jpg` — Default OG image for the landing page (1200×630)

Until these are replaced, the CSS gradient fallbacks will be used.
